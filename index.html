Installierte Module

Aktiviere oder deaktiviere installierte Game-Module im Hub. (Das Basis-Modul "Minecraft" ist immer aktiv.)

sports_esports

Willkommen im GameServer Hub

Dein zentrales Dashboard für alle Docker-basierten Gameserver auf Unraid. Verwalte deine Container, installiere neue Spiele mit einem Klick und bearbeite Dateien direkt im Browser.

extension

Game Additions

Egal ob Minecraft, CS2 oder Palworld – die Game Additions Module bieten dir maßgeschneiderte Dashboards für jedes Spiel. Lade Mods herunter, verwalte Spieler und passe Server-Properties an.

rocket_launch
